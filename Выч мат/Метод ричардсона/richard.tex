	\section*{Алгоритм}
	Найти минимальное и максимальное собственные значения матрицы:
	\[a = min(\lambda_i(A)), b = max(\lambda_i(A))\]	
	\[\tau^{0}_i = \dfrac{2}{(a + b) + (b-a)\cos\left(\dfrac{(2i-1)\pi}{2n}\right)},~~~i=1,2,...,k; \]
	\[x^{k+1} = x^k = \tau_{k+1}Ax^k + \tau_{k+1}f\]

	\end{pmatrix}\]
